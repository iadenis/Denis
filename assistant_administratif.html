Comment scanner depuis votre mobile :

  1. iPhone / iPad — Ouvrez l'app Notes, créez une note, appuyez sur 📷 → Scanner des documents. Exportez en PDF.
  2. Android — Ouvrez Google Drive, appuyez sur ＋ → Scanner. Le document est sauvegardé en PDF dans votre Drive.
  3. Application dédiée — Adobe Scan (gratuit) ou Microsoft Lens sont disponibles sur iOS et Android pour des scans de meilleure qualité.
  4. Une fois le PDF obtenu, revenez ici et utilisez Upload fichier pour l'importer.
📱 Adobe Scan iOS 🤖 Adobe Scan Android 🔷 Microsoft Lens

Comment scanner depuis votre imprimante / scanner bureau :

  1. Placez le document face vers le bas sur la vitre du scanner.
  2. Ouvrez le logiciel fourni avec votre imprimante (ex : HP Smart, Canon IJ Scan Utility, Epson Scan).
  3. Sélectionnez Scanner un document et choisissez le format PDF ou JPEG.
  4. Sur Windows : cherchez Télécopieur et scanner Windows dans le menu Démarrer — aucun logiciel tiers nécessaire.
  5. Sur Mac : ouvrez Capture d'image (dans Applications) pour scanner directement.
  6. Sauvegardez le fichier, puis importez-le avec Upload fichier.
